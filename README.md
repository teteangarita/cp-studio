# CP Studio — Consultoría de Servicios TI & Arquitectura Cloud

Bienvenidos al repositorio oficial de **CP Studio**, una firma de consultoría boutique premium especializada en ingeniería en la nube, optimización avanzada de bases de datos y desarrollo de software de alta gama.

Fundada y dirigida por **Carmen Perdomo**, CP Studio destaca en la industria por su filosofía única: la **fusión de máxima precisión técnica y rigor de ingeniería con una cercana y atenta conexión humana**.

---

## 🚀 Arquitectura y Servicios Especializados

En CP Studio diseñamos y desplegamos soluciones B2B estructuradas bajo estándares internacionales de alto rendimiento:

1.  **Arquitectura Cloud & DevOps:** Diseño de infraestructura escalable como código (IaC) con Terraform y orquestación con Kubernetes (EKS/GKE) en nubes como AWS y Google Cloud.
2.  **Optimización de Bases de Datos & Alta Carga:** Tuning de rendimiento crítico, control de bloqueos e indexación en PostgreSQL clústeres de alta concurrencia.
3.  **Desarrollo de Software a la Medida:** Ingeniería de aplicaciones transaccionales ágiles, backends robustos y tableros interactivos (dashboards).
4.  **Auditoría de Ciberseguridad & DevSecOps:** Blindaje perimetral API, análisis de vulnerabilidades, configuración de WAF y cumplimiento normativo PCI-DSS.

## 🎨 Manual y Paleta Visual (Branding)

El diseño visual del portal e interfaces se construye a partir del análisis digital y refinamiento del logotipo de la marca:

*   **Cobre Cálido (`#b07050`):** Color principal del logo. Transmite durabilidad, calidad artesanal, y cercanía humana.
*   **Espresso Profundo (`#251f1c`):** Neutro oscuro. Representa la solidez técnica, seguridad cibernética y elegancia.
*   **Arena Alabastro (`#fdfbf7`):** Fondo dominante. Brinda limpieza visual, frescura y excelente legibilidad digital.
*   **Gris Salvia (`#5f7470`):** Acento frío/mineral. Utilizado para iconografía de nube y balances técnicos.
*   **Durazno Suave (`#d8a47f`):** Color secundario de apoyo para micro-animaciones y degradados de transición.

## 📂 Estructura del Repositorio

*   **`index.html`**: Sitio web comercial oficial y portafolio interactivo de la consultora (incluye casos de éxito dinámicos y formulario de contacto).
*   **`brandbook.html`**: Portal interactivo del Manual de Identidad de Marca (directrices tipográficas, logo safe area, firma de correo HTML copiable y variables CSS).
*   **`style.css`**: Hoja de estilos unificada con el sistema de tokens, responsive grid y efectos premium.
*   **`script.js`**: Controlador javascript de interacciones, filtros de portafolio, lightbox modals y efectos de sparkles.
*   **`logo.jpeg`**: Logotipo corporativo de CP Studio.
*   **`.gitignore`**: Exclusión de archivos temporales del sistema.

## 💻 Despliegue Local y Visualización

Al ser una plataforma estática de alto rendimiento construida con **HTML5, CSS3 y JavaScript Vanilla**, no requiere de procesos complejos de compilación:

1.  Clona este repositorio:
    ```bash
    git clone https://github.com/tu-usuario/cp-studio.git
    ```
2.  Navega a la carpeta del proyecto:
    ```bash
    cd cp-studio
    ```
3.  Abre el archivo `index.html` en tu navegador favorito para visualizar la web comercial.
4.  Abre `brandbook.html` para consultar el manual interactivo de la marca.
